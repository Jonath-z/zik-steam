# Zik-Stream

This is a decentralized music streaming application that allows artist to be paid on each artist's song streamed.

## Tecnhologies

### Frontend
 
 - Next.js
 - Ethers.js

### Backend
 
 - Next.js
 - MongoDB Atlas
 
### Smart contract

- Solidity
- HardHat
- @openzeppelin 

# Note

The smart contrack is deployed on Rinkeby Test Network
